How to Setup & Use

User Mode
Tester Mode

User Mode - Quick Start

1

Download & Install

Download the latest release from GitHub and install the required dependencies:

pip install -r requirements.txt
2

Setup Environment

Create a .env file in the project root and add your GROQ API key:

GROQ_API_KEY=your_api_key_here
3

Configure Settings

Set up your social media accounts and preferences in the settings panel.

4

Start Scraping

Run the application and let Sniply collect only the posts worth your time:

python main.py

Tester Mode - Development Setup

1

Clone Repository

Clone the project and set up the development environment:

git clone https://github.com/khaled-muhammad/sniply.git
2

Install Dependencies

Install development dependencies and set up virtual environment:

python -m venv venv && source venv/bin/activate pip install -r requirements.txt
3

Setup Environment

Create a .env file in the project root and add your GROQ API key:

GROQ_API_KEY=your_api_key_here
4

Start Development

Run in development mode with debugging enabled:

python demo_mode.py
5

Choose Demo Mode

Select demo mode options (1 on, 2 off, 3 status): Enter 1 to turn on the mode:

1
6

Run Main & Start Scraping

Execute the main script and select option 1 to start scraping:

python main.py
